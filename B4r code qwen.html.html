Registration Overview

TIMU provides two types of registration through a simple, secure process that works on any phone via USSD *284#.

Driver Registration
Vehicle Registration

Driver Registration Process

  1. Dial *284# from any phone
  2. Select "Register as Driver"
  3. Enter your license information
  4. Create a 4-digit security PIN
  5. Confirm your registration via SMS

Vehicle Registration Process

  1. Dial *284# from any phone
  2. Select "Register Vehicle"
  3. Enter your vehicle information
  4. Create a 4-digit security PIN
  5. Confirm your registration via SMS

System Catalog

TIMU Rwanda System Catalog

The TIMU Rwanda Vehicle Registration System is a comprehensive platform designed to streamline vehicle and driver registration processes in Rwanda. This catalog provides detailed information about the system's components, capabilities, and integration points.

Vehicle Registration Module

Core Function RRA Integration USSD Support

Handles all aspects of vehicle registration including new registrations, renewals, and modifications. Integrated with Rwanda Revenue Authority (RRA) systems for seamless data exchange.

Key Features: Plate number assignment, owner information management, insurance status tracking, inspection scheduling

Data Sources: RRA vehicle database, insurance providers, inspection centers

Driver Management Module

License Management Compliance Tracking PIN Security

Manages driver information, license details, and compliance status. Supports multiple license classes and includes security features with PIN protection.

Key Features: License verification, violation tracking, renewal notifications, training record management

Integration Points: National identification system, license authorities, training providers

Fee Management System

Payment Processing Revenue Tracking Multi-channel

Handles all fee calculations, payment processing, and revenue tracking. Supports multiple payment methods including mobile money, bank transfer, and direct payment.

Fee Structures: Based on vehicle type, engine capacity, usage (commercial/private), and license class

Supported Payment Methods: Mobile money (MobiCash, MTN Mobile Money), bank transfer, cash at designated centers

Reporting & Analytics Engine

Data Visualization Custom Reports Export Capabilities

Provides comprehensive reporting capabilities with visualizations, trend analysis, and export functions. Supports regulatory reporting and operational analytics.

Report Types: Registration statistics, revenue collection, compliance rates, vehicle type distribution, regional analysis

Export Formats: PDF, Excel, CSV

USSD Integration

Mobile Access Offline Support SMS Integration

Enables vehicle and driver registration via USSD code *284# from any mobile phone. Does not require internet access, making it accessible nationwide.

Process Flow: Menu-driven USSD interface, data collection via SMS, confirmation and PIN setup

Telecom Partners: MTN Rwanda, Airtel Rwanda, and other mobile network operators

External System Integrations

RRA API Insurance Providers Banking Systems

Connects with external systems including Rwanda Revenue Authority, insurance companies, banking institutions, and mobile money providers.

Integration Methods: REST APIs, SFTP file transfer, web services

Data Exchange: Registration information, payment status, compliance data, owner information
